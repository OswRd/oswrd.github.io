ZIPs used in this tutorial

PixelSetupWizard-signed.zip by NikGApps team (can be found here)

  1. After you install GApps package, transfer PixelSetupWizard-signed.zip to internal storage



  2. Do a format data to avoid data encryption (and a bootloop because of that)


    3. At this time you can continue with step #3 below (if you don't want any kernel or magisk) or skip step #8 at ROMs AOSP installation (are the same as step #2 listed up) anc continue with step #9 at ROMs AOSP installation to install any kernel or Magisk.

  3. Reboot to system
