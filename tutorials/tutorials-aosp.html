ROMs AOSP installation

ZIPs used in this tutorial

ROM: Lucid v1.0 build 20200427 Queen Cake ---NO GAPPS PACKAGES ARE INCLUDED IN THIS ROM---

For root: Magisk v20.4

Kernel: Quantic kernel for Cepheus r8 by @Ayrton990


---PLEASE READ ALL THIS TUTORIAL BEFORE START WITH ANY INSTALLATION---

  1. Boot in TWRP


  2. Transfer to the internal store and flash latest firmware/vendor (can be found here)

    ---IF YOU ARE CURRENTLY ON LATEST MIUI ROM (EU, ESTABLE, MIUIMIX,ETC) IS NOT NECESSARY TO INSTALL FIRMWARE/VENDOR ZIP. YOU CAN SKIP THIS STEP---

  3. Move your ROM to any folder in internal storage



  4. Go to Install and select your ROM to flash (is not necessary to do wipes before flash the ROM) then swipe the bar to flash



  5. The last step is to do a format data to avoid data encryption (and a bootloop because of that)



  6. Reboot to system

    7. If your ROM does not include GApps and if you want to install them

  7. Tranfer your GApps package to the internal storage (smaller packages are recommended for any ROM)

    ---KEEP IN MIND THAT WITH SMALLER PACKAGES YOU WILL NOT BE ABLE TO HAVE SETUP WIZARD AND GOOGLE BACKUP SERVICES. IF YOU WANT TO INSTALL THEM CLICK HERE AFTER YOU FLASH YOUR GAPPS PACKAGE---

  8. If you don't care about the services listed in step #7 you can proceed to do a format data to avoid data encryiption

  9. Reboot to system

    10. If you want to install any kernel along with magisk

    ---A REBOOT IS NECESSARY AFTER INSTALL GAPPS/SERVICES PACKAGES TO CONTINUE TO INSTALL ANY KERNEL OR MAGISK---

  10. Transfer kernel.zip and magisk.zip to your internal storage now



  11. Go to install and select and flash magisk.zip first, once flashing ends will look like third image



  12. Now go to install again and select and flash your kernel.zip, once flashing ends will look like third image



  13. Wipe Cache/Dalvik is always optional



  14. Reboot system

    15. If you only want to install any kernel but no magisk

    ---A REBOOT IS NECESSARY AFTER INSTALL GAPPS/SERVICES PACKAGES TO CONTINUE TO INSTALL ANY KERNEL OR MAGISK---

  15. You have to transfer kernel.zip and magisk.zip to your internal storage now



  16. Go to install and select and flash magisk.zip first, once flashing ends will look like third image



  17. Now go to install and select and flash your kernel.zip, once flashing ends will look like third image



  18. Do a format data



  19. Wipe Dalvik/ART Cache and Cache



  20. Reboot system


  21. System is ready to use
