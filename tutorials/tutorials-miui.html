ROMs MIUI

ZIPs used in this tutorial

ROM: Xiaomi EU Weekly MIUI 12 Android 10 v20.5.7

For root: Magisk v20.4

Kernel: Quantic kernel for Cepheus r8 by @Ayrton990

  1. Boot in TWRP


  2. Move your ROM to any folder in internal storage



  3. Go to Install and select your ROM to flash (is not necessary to do wipes before flash the ROM)



  4. Swipe the bar to confirm flashing once you selected your ROM, once the process ends will look like this



  5. The last step is to do a format data to avoid data encryption (and a bootloop because of that)



  6. Reboot to system

    7. If you want to install any kernel along with magisk

  7. Transfer kernel.zip and magisk.zip to your internal storage now



  8. Go to install and select and flash magisk.zip first, once flashing ends will look like third image



  9. Now go to install again and select and flash your kernel.zip, once flashing ends will look like third image



  10. Wipe Cache/Dalvik is always optional



  11. Reboot system

    12. If you only want to install any kernel but no magisk

  12. You have to transfer kernel.zip and magisk.zip to your internal storage now



  13. Go to install and select and flash magisk.zip first, once flashing ends will look like third image



  14. Now go to install and select and flash your kernel.zip, once flashing ends will look like third image



  15. Do a format data



  16. Wipe Dalvik/ART Cache and Cache



  17. Reboot system
